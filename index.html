See every property change on every HubSpot record

Property Pulse is a HubSpot CRM card that surfaces the full change history of the properties that actually matter to your team -- directly on every record, with source, timestamp, and author for every edit.

Read the Setup Guide

What it does

Live property change history

Every tracked property gets a tile on the record showing the current value, previous value, and a reverse-chronological change log pulled straight from HubSpot's property history.

Full source and author detail

Each change log entry shows the source (CRM UI, Workflow, API, Integration, Import) and the name of the user who made the edit -- resolved automatically from the HubSpot owner directory.

Works with every object type

Supports Contacts, Companies, Deals, Tickets, and any custom objects in your portal. Custom objects are discovered automatically -- no per-object configuration needed.

Admin-configured tracked properties

Portal admins pick which properties to track per object type, set the card's layout column count, and control per-property display options -- all from a single settings page.

Personal tracked properties

Admins can let individual users add their own tracked properties on top of the admin defaults. Personal selections are scoped to the adding user only -- they never affect anyone else in the portal.

Paginated change log tables

Property history is rendered as a structured, sortable, paginated table modeled after HubSpot's own property history panel -- so long-lived records don't become endlessly scrolling walls of text.
